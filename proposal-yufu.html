FP基本方針

ファイナンシャルプランニング 5つの基本方針

  1. 年収500万円を絶対安全基準として設計
    一時的な年収800万円ではなく、安定収入ベースで無理のない返済計画を策定します。
  2. フルローンにより150万円の自己資金を緊急予備資金として確保
    頭金を入れず、手元資金を生活防衛資金として温存します。
  3. 金利1.5%上昇まで返済比率25%以内を維持
    変動金利リスクに対し、十分なバッファを持たせた設計です。
  4. 住宅ローン控除273万円（13年）を最大活用
    税制優遇をフル活用し、実質的な住居コストを削減します。
  5. NISA月3万円の積立投資を購入後も継続
    住宅取得と資産形成の両立を実現します。
  6. 団信を死亡保障の主軸として活用し、医療・就業不能保険を新規取得
    現在生命保険未加入のため、住宅ローンで付帯する団信が第一の死亡保障となります。これに加え、医療保険・就業不能保険への加入を優先課題とします。

FP推奨アクションプラン

Phase 1: 購入前— 2026年中
  1. 医療保険・就業不能保険への新規加入（合計月12,000〜18,000円）— 現在未加入のため最優先。就業不能保険は単身者の生活防衛の要
  2. 物件選定: 3,500万円以内・東京都内・管理費1.5万以内の条件で絞り込み
  3. 住宅ローン事前審査— SBI / 楽天 / PayPay等3行比較で最有利条件を確保。団信の保障内容も必ず確認
  4. 諸費用（約105万円）の資金計画確認— 貯蓄150万円からの支出範囲を精査
Phase 2: 購入直後— 1年以内
  1. 初年度確定申告で住宅ローン控除申請— 初年度は確定申告が必須（2年目以降は年末調整）。273万円の控除を確実に取得
  2. 団信の保障内容を確認・記録— 死亡・高度障害時のローン免除条件を把握。就業不能特約付き団信も検討
  3. NISA継続— 月3万円を維持し、住宅取得と資産形成を両立
  4. 緊急予備資金150万円は保全— 諸費用支出後の残額を絶対に手付かずで維持
Phase 3: 中長期— 10年
  1. 年収800万円期間中は繰上返済検討— ただしNISA投資を優先（期待リターン > ローン金利）
  2. 残債3,000万以下になったら固定金利への借換検討— 金利上昇リスクの軽減
  3. 5年ごとに収支見直し— ライフステージの変化に合わせた計画修正
  4. 65歳完済後の生活費計画策定— 老後資金2,000万円問題への対応
